# Peptide–DNA Conjugates for Formation of Lipid Nanodiscs

Scripts for building, validating, and analyzing peptide–DNA nanodisc systems in MD simulations.

- Structure preparation (DNA–peptide–lipid merging, topology consistency)  
- Validation (sequence checks, lipid–peptide interactions)  
- Analysis (distance, tilt angle, comparison plots)  

Tools: Python, GROMACS, PyMOL
